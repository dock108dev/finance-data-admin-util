## Data Flow

### Price Ingestion (equiv. Sports Ingestion)
```
External APIs вҶ’ Scraper (normalize) вҶ’ fin_candles + fin_sessions
```

### Exchange Sync (equiv. Odds Sync)
```
Exchange APIs вҶ’ Synchronizer вҶ’ fin_exchange_prices + fin_arbitrage_work
```

### Signal Pipeline (equiv. EV Computation)
```
fin_candles + fin_exchange_prices + fin_sentiment вҶ’ Signal Detector вҶ’ fin_alpha_signals
```

### Analysis Generation (equiv. Game Flow)
```
fin_candles + fin_social_posts + fin_alpha_signals вҶ’ Pipeline Orchestrator вҶ’ fin_market_analyses
```
